🍔 Hamburger House

Aplicação fullstack para simulação de uma hamburgueria, com foco em organização de estrutura, interface moderna e experiência do usuário.

🚀 Sobre o projeto

O Hamburger House é um projeto desenvolvido com o objetivo de praticar desenvolvimento fullstack, separando claramente frontend e backend, além de aplicar boas práticas de UI com uma identidade visual temática 🍔.

🎯 Objetivo

Praticar arquitetura fullstack
Trabalhar com organização de pastas e escalabilidade
Criar interfaces modernas com foco em UX
Desenvolver um projeto para portfólio

🛠️ Tecnologias utilizadas

Frontend
React
Vite
TailwindCSS
TypeScript
Backend
Node.js
Express

📂 Estrutura do projeto

hamburger-house/
├── backend/
└── frontend/

✨ Funcionalidades

Interface moderna com tema hamburguer 🍔
Estrutura fullstack separada
Página 404 personalizada (tema “em preparo”)
Navegação entre páginas
Base pronta para expansão (pedidos, cardápio, etc.)

🌐 Acesse o projeto

👉 https://weverton-charles.github.io/hamburger-house/

⚙️ Como rodar o projeto

🔹 Frontend
cd frontend
npm install
npm run dev

🔹 Backend
cd backend
npm install
npm run dev
🚀 Deploy

O projeto é publicado utilizando GitHub Pages, através da build do frontend:

cd frontend
npm run build
npm run deploy
